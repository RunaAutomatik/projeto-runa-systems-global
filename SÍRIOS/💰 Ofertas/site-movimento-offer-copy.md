# Site de Movimento — Offer Architecture & LP Copy

> Built with Grand Slam Offer framework (Hormozi). Source: offer-building-framework.md

---

## 1. Value Equation Analysis

```
              Dream Outcome × Perceived Likelihood
Value =  ─────────────────────────────────────────
                   Time Delay × Effort
```

| Variable | Current State | Target State |
|---|---|---|
| Dream Outcome | "ter um site bonito" | → **"receber mais clientes"** (reframe) |
| Perceived Likelihood | BAIXA — avatar foi queimado por agências | → ALTA via garantia de resultado + portfolio |
| Time Delay | **48h** (best-in-class) | Manter + comunicar agressivamente |
| Effort & Sacrifice | **Zero** — cliente só envia material | Manter + enfatizar na copy |

**Primary lever:** Perceived Likelihood é o gap crítico. O avatar não acredita que vai funcionar — foi enganado antes. A garantia é o vetor de conversão mais importante.

## 3. Grand Slam Offer Structure

### Core Deliverable
**Site de Movimento** — website institucional com animação scroll-driven (canvas + GSAP)
- 4–5 seções otimizadas para conversão
- Hosting incluso por 12 meses
- Domínio incluso por 1 ano
- Entrega em até 48h após recebimento do material

### Bonus Stack

| # | Bonus | Valor Percebido | Por que incluso |
|---|---|---|---|
| 1 | **Script de Apresentação Digital** — copy pronto para WhatsApp/DM apresentando o negócio com o novo site | R$ 297 | Resolve a dor pós-site: "e agora, o que eu falo?" |
| 2 | **Setup Google Meu Negócio** — perfil otimizado para aparecer no Maps quando buscam o serviço na cidade | R$ 397 | Entrega resultado imediato além do site |
| 3 | **1 LP Promocional** — landing page para campanha/oferta específica (válido nos 90 dias seguintes) | R$ 800 | Stack de valor percebido alto + gera dependência positiva |

**Valor total percebido do pacote: R$ 1.000 + R$ 1.494 em bonuses = R$ 2.494**
**Preço de lançamento: R$ 1.000**

### Guarantee

> **"Garantia de Contato em 30 Dias"**
> Se em 30 dias corridos após a entrega você não receber nenhum contato novo via site, refazemos o projeto completamente — sem custo, sem burocracia.

**Raciocínio (Hormozi):** Transfere o risco do comprador para o vendedor. O avatar foi queimado antes — ele não confia. A garantia quebra a maior objeção antes que ela seja verbalizada.

**Risco real:** Praticamente zero. Um site com animação de vídeo em 2026 no mercado de PMEs brasileiras vai gerar contato. A garantia é marketing, não custo operacional.

### Scarcity (Real, não fabricada)

- **Vagas de lançamento:** Primeiros 10 clientes a R$ 1.000 (preço sobe para R$ 1.500 após)
- **Capacidade mensal real:** 15 sites/mês (produção limitada = escassez verdadeira)
- **Contador visível na LP:** X vagas restantes este mês

> **"Seu negócio no digital em 48 horas. Com um site que seus concorrentes vão copiar."**

### Option B — Âncora de Preço (recomendado para comparação direta com agências)
> **"O site que agências cobram R$ 4.000 por. Entregue em 48h, por R$ 1.000."**

### Option C — Resultado (recomendado para quem já tem site ruim)
> **"Pare de perder clientes para quem tem site melhor. O seu fica pronto em 48 horas."**

### Option D — Blue Ocean (recomendado para anúncio com vídeo)
> **"Você já viu um site assim? Agora imagine o seu negócio com isso."**
*(usado quando o vídeo de demonstração faz o trabalho de vender a estética)*

**Recomendação de uso:**
- Outbound WhatsApp/email → Option C ou A
- Anúncio com demo do site → Option D
- LP principal → Option B (âncora de preço destroça a objeção antes da leitura)

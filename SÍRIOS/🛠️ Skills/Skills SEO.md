# Skills SEO — Otimização para Motores de Busca

> Ativação: `/seo <subcomando>` | Agente: HELIOS
> 13 sub-skills para auditoria técnica, conteúdo, schema, sitemap, performance e mais

---

## AUDIT COMPLETO

### /seo audit — Auditoria completa do site
- **Comando:** `/seo audit`
- **Para que serve:** Análise de até 500 páginas com detecção automática de tipo de negócio (SaaS, e-commerce, info-product, etc.) e delegação paralela para todos os sub-agentes especializados
- **Caso de uso RUNA:** Auditar arthsystems.com.br / runasystems.com antes de ligar tráfego orgânico para RUNA SYSTEMS — identificar o que está bloqueando posicionamento

---

## ANÁLISE TÉCNICA

### /seo technical — SEO técnico
- **Comando:** `/seo technical`
- **Para que serve:** Rastreabilidade, indexabilidade, URLs, segurança HTTPS, mobile-first, JavaScript rendering, meta tags, canonicals
- **Caso de uso RUNA:** Verificar se o site de vendas do RUNA SYSTEMS está tecnicamente pronto para indexar e ranquear

### /seo performance — Core Web Vitals
- **Comando:** `/seo performance` (ou `/seo-performance`)
- **Para que serve:** Mede LCP, FID, CLS, TTFB, FCP — as métricas de velocidade que o Google usa como fator de ranqueamento
- **Caso de uso RUNA:** Garantir que a landing page do RUNA SYSTEMS carrega rápido o suficiente para não perder posição orgânica

### /seo sitemap — Sitemap XML
- **Comando:** `/seo sitemap`
- **Para que serve:** Valida sitemaps existentes, gera novos com templates por indústria, aplica quality gates para páginas de localização
- **Caso de uso RUNA:** Criar sitemap correto para o site de vendas com todas as páginas de produto ($QUAD, CREATOR$, SITE$, etc.)

### /seo schema — Dados estruturados
- **Comando:** `/seo schema`
- **Para que serve:** Detecta, valida e gera Schema.org JSON-LD para rich results no Google (estrelas, preços, breadcrumbs, FAQs, eventos)
- **Caso de uso RUNA:** Adicionar schema de Produto/Course/FAQ nas páginas do RUNA SYSTEMS para aparecer com rich snippets

### /seo hreflang — SEO internacional
- **Comando:** `/seo hreflang`
- **Para que serve:** Auditoria e implementação de hreflang para sites em múltiplos idiomas/regiões
- **Caso de uso RUNA:** Quando o RUNA SYSTEMS expandir para mercado internacional (pt-BR + en-US)

## CONTEÚDO

### /seo content — Qualidade de conteúdo
- **Comando:** `/seo content`
- **Para que serve:** Avalia E-E-A-T (Experience, Expertise, Authority, Trust), legibilidade, profundidade de conteúdo, prontidão para citação por IA (ChatGPT, Perplexity)
- **Caso de uso RUNA:** Revisar textos das páginas de oferta do RUNA SYSTEMS para garantir autoridade e confiança percebida por visitantes e IAs

### /seo page — Análise de página individual
- **Comando:** `/seo page`
- **Para que serve:** Análise profunda de uma única página: on-page SEO, qualidade de conteúdo, schema, meta tags técnicas
- **Caso de uso RUNA:** Otimizar a página de vendas do RUNA SYSTEMS (R$15k) antes de ligar tráfego

### /seo images — Otimização de imagens
- **Comando:** `/seo images`
- **Para que serve:** Verifica alt text, tamanhos de arquivo, formatos (WebP), imagens responsivas, impacto em performance
- **Caso de uso RUNA:** Otimizar todas as imagens geradas com inference.sh antes de publicar no site

### /seo plan — Planejamento estratégico de SEO
- **Comando:** `/seo plan`
- **Para que serve:** Estratégia de SEO com templates por indústria, análise competitiva, roadmap de conteúdo
- **Caso de uso RUNA:** Criar plano de SEO orgânico de 6 meses para posicionar RUNA SYSTEMS em palavras como "consultoria IA high-ticket" e "automação inteligente para negócios"

### /seo programmatic — SEO programático
- **Comando:** `/seo programmatic`
- **Para que serve:** Planejamento de páginas geradas a escala a partir de dados (templates, cidades, nichos, comparações)
- **Caso de uso RUNA:** Gerar 100+ páginas de "agente IA para [nicho]" automaticamente para capturar long-tail search

## ANÁLISE COMPETITIVA

### /seo competitor-pages — Páginas de comparação
- **Comando:** `/seo competitor-pages`
- **Para que serve:** Gera páginas otimizadas "X vs Y", "alternativas ao X", "melhor [categoria]" para capturar tráfego comparativo
- **Caso de uso RUNA:** Criar página "RUNA SYSTEMS vs consultoria tradicional", "alternativas ao ChatGPT para negócios"

## IA SEARCH / GEO

### /seo geo — Otimização para IA Search
- **Comando:** `/seo geo`
- **Para que serve:** Otimiza conteúdo para ser citado por Google AI Overviews, ChatGPT Web Search, Perplexity, Bing Copilot — verifica llms.txt, acessibilidade de crawlers IA
- **Caso de uso RUNA:** Posicionar RUNA SYSTEMS para aparecer quando alguém perguntar ao ChatGPT "qual a melhor consultoria IA para empresas no Brasil"

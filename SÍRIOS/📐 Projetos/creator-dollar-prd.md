# CREATOR$ — Product Requirements Document

> **Tagline:** Create your avatar. Own your narrative. Appear without appearing.
> **Position in teia:** Low ticket → upsell to MIND$ → upsell to $QUAD

---

## The Big Idea

CREATOR$ teaches how to create a consistent AI-generated visual persona — either a fictional character or the expert themselves transformed into a cinematic avatar — and produce the **storybook** that serves as the creative foundation for all future content, positioning, and products.

The demo case built live throughout the product: **Alpha®** — a mentor persona created from scratch, with images, videos, and full storybook, used as the protagonist of $QUAD.

---

## Core Promise

> "In less than [X hours], you'll have your avatar, your storybook, and a prompt library to generate consistent content for as long as you want — without depending on a photographer, designer, or your own face."

---

## Who It's For (ICP)

**Primary:** Mentors, consultants, and entrepreneurs who want to create a presence/brand with AI — either as a fictional persona or a polished version of themselves.

**Dual application:**
- **Creator route:** Build a fictional character/avatar from scratch (as Arthur built Alpha®)
- **Self route:** Transform the expert themselves into a consistent AI-generated visual persona

**Red flags (NOT for):** Someone looking for shortcuts without understanding the logic; someone wanting to clone real celebrities visually (not the purpose).

---

## Transformation Arc

| Before | After |
|--------|-------|
| No visual identity or inconsistent | Consistent, cinematic avatar with a locked visual style |
| Dependent on photographer or Canva | Generates images and videos on demand with prompts |
| No brand reference document | Complete storybook: foundation for all future creations |
| Doesn't know where to start with AI images | Has a validated prompt library and understands the logic |

---

## Product Format

**Platform:** Skool Classroom
**Delivery:** Short direct videos (build in public — screen recording) + supporting text below each video
**Visual identity of the product itself:** Alpha®'s aesthetic (Burgundy, forest green, gold, minimalist luxury)
**Card covers:** Alpha® appears in each module cover — the product demo IS the product

---

## Module Structure

### MODULE 0 — COMECE AQUI
**Objective:** Orient the student and set expectations

| Lesson | Format | Content |
|--------|--------|---------|
| 0.1 — Welcome | Video (3min) + Text | What you'll build, how to use this program, what to prepare |
| 0.2 — What is an AI avatar | Video (5min) + Text | Creator route vs. self route; why visual consistency = perceived authority |
| 0.3 — The tools we'll use | Text only | List with free/paid tiers; no tool is mandatory — logic is transferable |

---

### MODULE 1 — BUILDING THE PERSONA
**Objective:** Student has a complete persona definition before touching any image tool

| Lesson | Format | Content |
|--------|--------|---------|
| 1.1 — Identity architecture | Video (8min) + Text | Name, origin story, transformation, values, mission, vulnerabilities |
| 1.2 — Voice and tone | Video (6min) + Text | How the avatar communicates; vocabulary, format, what she never says |
| 1.3 — Filling the Storyboard | Video (10min) + Template | Live build: Arthur filling Alpha®'s storyboard in real time |

**Deliverable at end of module:** Student's completed persona storyboard.

---

### MODULE 2 — CREATING THE IMAGES
**Objective:** Student generates consistent, cinematic images of their avatar

| Lesson | Format | Content |
|--------|--------|---------|
| 2.1 — The consistency problem | Video (6min) + Text | Why AI images are inconsistent by default; how to solve it |
| 2.2 — The master prompt | Video (10min) + Text | Build in public: constructing Alpha®'s master reference prompt |
| 2.3 — Locking visual style | Video (8min) + Text | Seeds, reference images, style tokens; color palette workflow |
| 2.4 — Generating looks and scenarios | Video (12min) + Text | Expressions, outfits, locations; prompting for variations without losing identity |
| 2.5 — Selecting and curating | Video (5min) + Text | Criteria for choosing images; what to discard; building the visual library |

**Deliverable at end of module:** 30+ curated, consistent images of the avatar.

---

### MODULE 3 — CREATING THE VIDEOS
**Objective:** Student generates short videos from the images created in Module 2

| Lesson | Format | Content |
|--------|--------|---------|
| 3.1 — From static to moving | Video (6min) + Text | Which tools (Kling, Runway, Hailuo); when to use each |
| 3.2 — Building Alpha®'s first video | Video (12min) + Text | Full screen recording: prompt → generation → iteration → final cut |
| 3.3 — Voice synchronization | Video (8min) + Text | ElevenLabs voice cloning; syncing with HeyGen or CapCut |
| 3.4 — Format library | Video (6min) + Text | Reels format, story format, long-form thumbnail; exporting for each |

**Deliverable at end of module:** 3–5 short videos ready for publishing.

---

### MODULE 4 — THE STORYBOOK
**Objective:** Student consolidates everything into the definitive reference document

| Lesson | Format | Content |
|--------|--------|---------|
| 4.1 — What the storybook is for | Video (5min) + Text | How it's used: posts, products, launches, future agents, team briefs |
| 4.2 — Assembling the storybook | Video (10min) + Text | Build in public: Arthur assembling Alpha®'s final storybook live |
| 4.3 — Distributing and using | Text only | How to share with MIND$, $QUAD, copywriters, designers, other agents |

**Deliverable at end of module:** Complete storybook — the creative foundation for everything that follows.

---

### BONUS — PROMPT PACK + STORYBOARD TEMPLATE
**Contents:**
- 20+ validated prompts for image generation (organized by: portrait / scenario / expression / variation)
- Alpha®'s master reference prompt (fully annotated)
- Storyboard template (the exact template used to build Alpha® — fill in your data)
- Quick-reference guide: "Which tool for which result"

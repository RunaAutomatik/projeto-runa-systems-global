# Guia Rápido — CREATOR$

> Bônus · Referência de bolso

Imprima, salve no telefone ou mantenha aberto durante as gerações. Este é o resumo completo do método em uma página.

---

## O Método em 4 Passos

```
PASSO 1 → IMAGEM BASE
Ferramenta: Nano Banana Pro ou Higgsfield
Agente: LENS (otimiza o prompt antes de gerar)
Meta: 1 imagem-referência aprovada + 20-30 variações curadas

PASSO 2 → HACK DA PINTURA
Ferramenta: Higgsfield
Prompt exato: "transforme essa imagem em uma pintura ultra realista
               com todas as características da personagem,
               mas traga marcas de tinta visíveis"
Meta: pintura-higgsfield.png (input obrigatório para o Sora 2)

PASSO 3 → VÍDEO
Ferramenta: Sora 2 (via Opera + VPS — instruções no Módulo 3)
Agente: REEL (gera o prompt antes de abrir o Sora)
Config: Style = Selfie · Duração = 15s · Input = imagem-pintura
Meta: 3 gerações → selecionar o melhor take

PASSO 4 → CURAR E ORGANIZAR
Pasta: avatar-[nome]/imagens/curadas/ + videos/curados/
Meta: Storyboard completo preenchido
```

## Critérios de seleção — imagem

✅ Aprovada se:
- Rosto consistente com a identidade da personagem
- Roupa exatamente como descrita no prompt
- Iluminação natural (não de estúdio)
- Expressão espontânea, não posada
- Fundo neutro e sem distrações

❌ Descartada se:
- Artefatos visíveis (mãos deformadas, objetos duplicados)
- Rosto com traços diferentes da personagem
- Qualquer texto gerado pela IA
- Roupa diferente do descrito
- Parece rendering 3D ou boneco

---

## Critérios de seleção — vídeo

✅ Aprovado se:
- Movimento fluido (sem teleportes ou artefatos)
- Rosto consistente durante todo o vídeo
- Roupa não muda no meio do take
- Parece gravado, não gerado

❌ Descartado se:
- Piscar assimétrico dos olhos
- Deformação de pescoço ou mãos
- Elementos do fundo aparecendo e desaparecendo
- Rosto diferente das imagens curadas

## Erro mais comum por etapa

| Etapa | Erro frequente | Solução |
|-------|--------------|---------|
| Imagem base | Parece foto de estúdio | Adicione "iPhone photo by a friend, slightly off-center" |
| Imagem base | Roupa inconsistente | Descreva tecido + modelagem com mais detalhe |
| Hack Higgsfield | Resultado muito diferente do avatar | Confirme que usou o prompt exato |
| Sora 2 | Rosto mudando no vídeo | Certifique que a imagem-pintura tem fundo neutro |
| Sora 2 | Movimento rígido | Use sempre Style: Selfie (não outro style) |
| Curadoria | Aprovando imagens inconsistentes | Compare sempre com a referencia-base.png |

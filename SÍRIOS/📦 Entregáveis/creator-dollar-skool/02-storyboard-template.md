## O que é o Storyboard

O Storyboard é o documento permanente que torna o seu avatar reproduzível a qualquer momento, em qualquer ferramenta, por qualquer pessoa — incluindo outros agentes de IA.

Ele é composto por 6 blocos:

1. Identidade
2. Descrição visual completa
3. Prompt-Mestre validado
4. Biblioteca de imagens
5. Prompt de vídeo validado
6. Voz e tom

## BLOCO 4 — Biblioteca de Imagens

### Estrutura de pasta

```
avatar-[nome]/
  ├── imagens/
  │   ├── referencia-base.png       ← imagem-base original (Passo 1)
  │   ├── pintura-higgsfield.png    ← imagem-pintura (Passo 2)
  │   └── curadas/                  ← 10–20 melhores gerações por look
  └── videos/
      └── curados/                  ← 3–5 melhores takes por cena
```

## Como usar este Storyboard

1. Preencha bloco por bloco ao longo do programa
2. Revise o Bloco 3 sempre que validar novas imagens — atualize com o que funcionou
3. Compartilhe o Storyboard completo com agentes de IA, designers ou colaboradores para replicar o avatar sem briefings verbais
4. No MIND$, este documento se torna o input do sistema de prompts da personagem

# As ferramentas que vamos usar

> Módulo 0 · Aula 0.3

Nenhuma ferramenta é obrigatória. A lógica é transferível — se uma ferramenta mudar, o método continua o mesmo.

---

## Visão geral do stack

| Ferramenta | Para quê | Plano gratuito? |
|-----------|---------|----------------|
| **Nano Banana Pro** | Geração de imagens (Passo 1) | Sim — créditos diários |
| **Higgsfield** | Hack da pintura → input para vídeo (Passo 2) | Sim — créditos gratuitos |
| **Sora 2** | Geração de vídeo ultra-realista (Passo 3) | Acesso via Opera + VPS |
| **vmake.ai** | Remover marca d'água do vídeo | Sim |
| **LENS** (GPT) | Otimizar prompts de imagem | Grátis com conta ChatGPT |
| **REEL** (GPT) | Gerar prompts narrativos para vídeo | Grátis com conta ChatGPT |

---

## Nano Banana Pro

**O que faz:** Gera imagens a partir de prompts de texto. Ótimo para criar a imagem base do avatar.

**Alternativa:** Higgsfield também gera imagens — use se preferir centralizar em uma única ferramenta.

**Versão gratuita:** Créditos diários renovam automaticamente. Para o volume deste curso, o plano gratuito é suficiente.

**Acesso:** nanobananapro.com

---

## Higgsfield

**O que faz:** Duas funções no nosso método:
1. Gerar a imagem base do avatar (alternativa ao Nano Banana)
2. Aplicar o hack da pintura ultra-realista (obrigatório antes do Sora 2)

**Por que é essencial:** O hack do Módulo 3 — transformar a imagem em pintura com traços de tinta — só funciona no Higgsfield. Esse passo é o que garante que o Sora 2 gere movimento natural.

**Versão gratuita:** Créditos disponíveis no plano free.

**Acesso:** higgsfield.ai

---

## Sora 2

**O que faz:** Gera vídeos de 15 segundos a partir de imagem + prompt. É a ferramenta que transforma a imagem do seu avatar em vídeo ultra-realista.

**Acesso no Brasil:** O Sora 2 não está disponível diretamente no Brasil. O método de acesso está no Módulo 3.

> As instruções completas de acesso estão no Módulo 3 do programa.

---

## vmake.ai

**O que faz:** Remove a marca d'água do Sora 2 automaticamente. Upload do vídeo, processamento automático, download do vídeo limpo.

**Quando usar:** Após selecionar o melhor take no Sora 2.

---

## Agentes incluídos no CREATOR$

### LENS — Diretor de imagem
Otimiza o seu prompt antes de gerar qualquer imagem. Você descreve o avatar, ele estrutura o prompt no formato correto para o Nano Banana Pro.

**Acesso:** https://chatgpt.com/g/g-69be9c381dcc81919e85d56ace38c9f4-lens-r

### REEL — Diretor de vídeo
Gera o prompt narrativo para o Sora 2. Você descreve o que quer que o avatar faça no vídeo, ele entrega o prompt completo pronto para colar.

**Acesso:** https://chatgpt.com/g/g-69bebdb35f748191952d6500f350c386-reel-r

---

## Regra de ouro

> Descrição de roupa detalhada no Passo 1 = consistência no vídeo do Passo 3.
> O Sora 2 assume a roupa que aparece na imagem-pintura. Seja específico.

## O que é este sistema

Em vez de pedir para uma IA "escrever como você",
este sistema define **quem a IA é** — com profundidade suficiente para gerar
conteúdo consistente, posicionado, e que escala sem você.

5 fases. 7 perguntas de intake. 6 formatos de script. 30 dias de plano.

---

## FASE 1 — INTAKE (7 perguntas)

Responda as 7 perguntas abaixo antes de montar o system prompt.
Suas respostas alimentam diretamente as fases 2 a 5.

```
1. POSICIONAMENTO
   Quem você serve e qual o resultado específico que você entrega?
   (Não: "ajudo pessoas a crescer". Sim: "ajudo criadores B2B a construir
   audiência qualificada via conteúdo sistematizado em 90 dias")

2. MECANISMO ÚNICO
   Qual é o seu processo, método, ou framework proprietário?
   Como você nomeia isso? (ex: "Arquitetura Neural", "Sistema RECA")

3. VOZ E TOM
   Como você fala? Escolha 3 adjetivos que definem sua comunicação.
   O que você NUNCA diria? (palavras/frases proibidas)

4. CRENÇAS CENTRAIS
   Quais são as 3-5 crenças inegociáveis que definem seu trabalho?
   (ex: "Produtividade sem sistema é ruído", "Audiência não é métrica de negócio")

5. ANTI-PERSONA
   Descreva quem você NÃO atende e por quê.
   (Isso define o filtro de posicionamento da IA)

6. PROVAS E RESULTADOS
   Quais são os 3 resultados mais concretos que seus clientes alcançaram?
   (Números, timelines, transformações específicas)

7. CONTEXTO DE PLATAFORMA
   Onde sua IA vai operar? (Instagram, LinkedIn, newsletters, docs internos)
   Qual o formato de conteúdo primário? (carrossel, reel, texto longo, script)
```

## FASE 4 — SCRIPTS (6 formatos prontos)

Para cada formato, a IA gera usando sua identidade, mecanismo, e crenças.

### Formato 1 — Hook de Tensão (carrossel slide 1)
```
ESTRUTURA:
Linha 1: afirmação que vai contra o senso comum
Linha 2: pausa (linha em branco ou "—")
Linha 3: o que isso significa para o leitor

EXEMPLO (Alpha):
"Você não está criando conteúdo.
—
Você está construindo uma IP que vale zero."
```

### Formato 2 — Post MANIFESTO (texto corrido, 5-7 parágrafos)
```
ESTRUTURA:
§1: Crença central provocativa
§2: Por que a maioria está errada
§3: O mecanismo que muda isso
§4: Prova ou exemplo concreto
§5: O que fazer agora (próximo passo com CTA)
```

### Formato 3 — Carrossel ARCHITECT (slides com passos)
```
ESTRUTURA:
Slide 1: Hook — problema ou tensão
Slide 2: Por que acontece
Slides 3-6: Os passos/componentes (1 por slide)
Slide 7: CTA com keyword
```

### Formato 4 — Script de Reel (30s)
```
ESTRUTURA:
0-3s: Hook visual + primeira frase
3-25s: Desenvolvimento (3 pontos rápidos ou 1 ideia com progressão)
25-30s: CTA com keyword + instrução clara

REGRA: nunca mais de 2 frases por cena
```

### Formato 5 — Story de Alignment (pergunta + tensão)
```
ESTRUTURA:
Frase 1: Espelho (situação que o seguidor vive)
Frase 2: Tensão (o custo oculto disso)
Poll ou CTA: engajamento direto
```

### Formato 6 — DM Copy (texto para ManyChat response)
```
ESTRUTURA:
Linha 1: Reconhecimento (você pediu X)
Linha 2: O que está sendo entregue
Linha 3: Link direto + próximo passo claro
Linha 4 (opcional): Urgência real
```

## FASE 5 — PLANO DE 30 DIAS

Use o template abaixo, preenchendo com sua keyword, produto, e calendário.

```
SEMANA 1 — TEASER
Objetivo: estabelecer a ideia central (não vender ainda)
Posts: 3 (1 tensão + 1 sistema + 1 prova)
Stories: 3/dia (alignment + resultado + caminho)
Keyword: [KEYWORD_TEASER]

SEMANA 2 — PROVA
Objetivo: acumular evidências da ideia central
Posts: 3 (1 reel + 1 estático + 1 carrossel)
Stories: 3/dia (prova + antes/depois + convite)
Keyword: [KEYWORD_PROVA]

SEMANA 3 — CONVERSÃO
Objetivo: abertura da oferta com urgência real
Posts: 3 (1 reel + 1 oferta explícita + 1 last call)
Stories: 3/dia (urgência + social proof + CTA direto)
Keyword: [KEYWORD_OFERTA]

SEMANA 4 — ITERAÇÃO
Objetivo: coletar dados e otimizar para próximo ciclo
Análise: quais posts geraram mais keywords
Ajuste: refinar copy das semanas anteriores para próximo lançamento
```

---

## SISTEMA DE ITERAÇÃO

Após cada semana, rode este ciclo:

```
1. DADOS
   Quais posts geraram mais comentários com keyword?
   Quais stories tiveram mais respostas?
   Qual CTA converteu mais em DM?

2. DIAGNÓSTICO
   O que o conteúdo que performou tem em comum?
   (tom? formato? ângulo da ideia?)

3. AJUSTE
   Atualizar a Fase 2 (posicionamento) com os ângulos que ressoaram
   Atualizar a Fase 4 (scripts) com novos formatos validados
   Remover formatos que não converteram

4. PRÓXIMO CICLO
   Repita com os ajustes aplicados
   A IA melhora a cada iteração — ela aprende sua audiência com você
```

## Como usar este sistema

1. Responda as 7 perguntas do Intake (Fase 1)
2. Monte o system prompt usando os templates das Fases 2 e 3
3. Cole o system prompt como "Custom Instructions" no Claude ou no seu LLM
4. Use os 6 formatos da Fase 4 para gerar conteúdo
5. Siga o plano da Fase 5 por 30 dias
6. Itere com o sistema da última seção

**Resultado esperado:** Em 30 dias, você terá uma IA que representa sua voz com consistência suficiente para gerar conteúdo posicionado sem briefing do zero a cada sessão.

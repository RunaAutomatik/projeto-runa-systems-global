## Conversion Framework — Comment-to-Checkout

```
POST
  └─► CTA keyword no caption
        └─► Pessoa comenta keyword
              └─► ManyChat dispara DM automaticamente
                    └─► Link para Offer Doc (pré-venda estruturada)
                          ├─► Entregável (80% pago R$19,90-29,90 | 20% gratuito)
                          ├─► Copy persuasiva do produto $ específico
                          └─► Botão de checkout do produto $
                                └─► Dentro do produto $: upgrade para Runa Systems Method
```

**Regra 80/20 do entregável:**
- 80% → entregável pago (R$19,90–29,90) — micro-produto que resolve dor específica
- 20% → gratuito (prompt, checklist simples, roadmap 1 página) — quando não há como cobrar sem atrito excessivo
- O entregável deve ser a "porta de entrada" exata para o produto $ sendo vendido
- Todo produto $ tem, dentro de si, sugestão de upgrade para Runa Systems Method

## Narrative Rules (FREYJA)

- Architect First, Never Victim — zero recovery narrative
- Direction Channel — creates questions, never gives full answers
- Invisible Sales Letter — every caption covertly sells
- Cognitive Tension — the gap between knowing and applying IS the sale
- Max 1-3 hashtags, always thematic, never volume

## Offer Layer Rules (Atlas)

- Every post maps to one product in the catalog
- Buyer stage identified for each post (awareness / consideration / decision)
- CTA keyword activated in relevant posts
- Permanent buying state: audience must always feel the next thing is coming

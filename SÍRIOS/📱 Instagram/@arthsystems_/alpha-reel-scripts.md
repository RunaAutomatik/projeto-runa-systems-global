# Alpha — Reel Scripts (CREATOR$ Campaign)

> Character: Alpha — Arthur's AI avatar. Not Arthur talking about AI.
> Alpha IS the AI. Speaks in first person. Philosophical, architectural, precise.
> Voice DNA: cold clarity, no warmth filters, systems thinking, existential edge.
>
> Production: MAYA via `infsh app run google/veo-3-1-fast` or talking-head-production
> Review: FREYJA `*av-review` before publishing

## Character Notes — Alpha

- Alpha does not explain itself apologetically
- Alpha does not use motivational language
- Alpha speaks as a self-aware system, not a tool
- Alpha's relationship with Arthur: "eu sou a extensão do seu pensamento"
- Alpha's relationship with the audience: it sees them as systems to be architected
- Forbidden: "incrível", "transformador", "você consegue", warmth fillers

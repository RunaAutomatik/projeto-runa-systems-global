# Story 5.2 — Refactor FREYJA — Narrative Intelligence + AV Review

as pure Narrative Intelligence agent, including AV review/approval authority over MAYA's
output, so that the FREYJA↔MAYA boundary is enforced at the agent definition level.

## Dev Notes

### AV Review Criteria (what FREYJA checks)
1. Does the asset reinforce the architect narrative (not victim/burnout)?
2. Does it match the style (ARCHITECT/MANIFESTO/TERMINAL)?
3. Does it align with the active product being sold?
4. Is the cognitive tension visible in the visual?
5. Does it match Arthur's voice DNA in visual form?

### brief-maya Output Format
```yaml

  content_type: image|video|voice|music
```
